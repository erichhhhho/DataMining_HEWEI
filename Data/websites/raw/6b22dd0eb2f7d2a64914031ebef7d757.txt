<p>Residents of Xinjiang, an ethnic minority region of western China, are being forced to install spyware on their mobile phones.</p>
<p>On July 10, mobile phone users in the Tianshan District of Urumqi City received a mobile phone notification from the district government instructing them to install a surveillance application called <a href="http://jw.js.vnet.cn/2017/faq.html">Jingwang</a> (or &#8220;Web Cleansing&#8221;). The message said the app was intended to &#8220;prevent [them] from accessing terrorist information.&#8221;</p>
<p>But authorities may be using the app for more than just counter-terrorism. According to an <a href="http://www.rfa.org/mandarin/yataibaodao/shaoshuminzu/ql2-07132017112039.html"> exclusive report</a> from Radio Free Asia, 10 Kazakh women from Ili Kazakh Autonomous Prefecture were arrested for messages sent to a private WeChat group chat soon after they installed the app.</p>
<p>The notification from police said the application would locate and track the sources and distribution paths of terrorists, along with &#8220;illegal religious&#8221; activity and &#8220;harmful information,&#8221; including videos, images, ebooks and documents.</p>
<p>Jingwang&#39;s website describes the application as follows:</p>
<blockquote><p>净网卫士是江苏电信面向家庭宽带用户推出的家长、小孩上网分级防护服务，主要提供色情网站拦截；网络诈骗、病毒木马、钓鱼网站阻断；上网时间控制提醒；手机远程管理等安全防护服务。为未成年人筑起一道网络安全屏障，为孩子的健康成长保驾护航。</p></blockquote>
<blockquote class="translation"><p>Jingwang is a protection service with an adult and child categorization system introduced by Jiangsu Telecom. The main function is to block pornographic websites, online scams, trojan horses, and phishing sites; to alert users of how much time they spend online; and to enable remote control of one&#39;s home network. The tool is intended to help kids develop a healthy lifestyle by building a safe web filter for the minors.</p></blockquote>
<p>Of course, any tool with these capabilities could be used in multiple ways. For example, the app&#39;s &#8220;remote control&#8221; feature could enable state actors or even hackers to manipulate or steal from a person&#39;s home network.</p>
<p>The move is consistent with other measures of control over digital activities in the region. While stories of digital censorship in China often focus on the experiences of users in major cities in the east and south, the reality is often <a href="https://advox.globalvoices.org/2013/12/17/in-chinas-ethnic-minority-regions-internet-blackouts-are-the-norm/">more bleak</a> for those living in remote, embattled ethnic minority regions such as Xinjiang and Tibet. Seeking to contain unrest and discontent in conflict areas, authorities often impose <a href="https://advox.globalvoices.org/2016/01/20/china-is-blocking-circumvention-tools-with-help-of-cloud-service-providers/">extreme censorship</a> and surveillance measures and routine <a href="https://advox.globalvoices.org/2013/12/17/in-chinas-ethnic-minority-regions-internet-blackouts-are-the-norm/">Internet shutdowns</a>.</p>
<p>A Twitter-based media outlet, “Images from mainland China” (即时中国大陆映像), which covers censored news in China, posted photos taken from a checkpoint where police officers randomly checked residents to see if they have installed the surveillance app:</p>
<blockquote class="twitter-tweet" data-lang="en">
<p dir="ltr" lang="zh">新疆各部手机检查安装净网卫士官方软件，短信通知十日内安装，临检发现没安装拘留十天，科技倒退，逼老百姓用老年机，什么叫禁锢，这种全面监控就是禁锢。回到长毛时代 <a href="https://t.co/zUnLXc9tFA">pic.twitter.com/zUnLXc9tFA</a></p>
<p>— 即时中国大陆映像✊✊✊ (@o66071443) <a href="https://twitter.com/o66071443/status/887107966368968704">July 18, 2017</a></p></blockquote>
<blockquote class="translation"><p>Authorities from Xinjiang are checking to make sure that people are using the official Jingwang application. A mobile notification demanded people install the app within 10 days. If they are caught at a checkpoint and their devices do not have the software, they could be detained for 10 days. This is a setback on the development of technology. They forced people to use devices designed for the elderly. It is a form of confinement by through surveillance technology. We are back to Mao&#39;s China.</p></blockquote>
<p>“Images from mainland China” (即时中国大陆映像) also posted a product description of Jingwang which explained that the tool can negate the password requirement of a Windows operating system and access the computer hard disk with no restrictions.</p>
<blockquote class="twitter-tweet" data-lang="en">
<p dir="ltr" lang="zh">新疆人民电脑手机，净网卫士强制安装后，相当于电子手铐 <a href="https://t.co/NIF4dffCIO">pic.twitter.com/NIF4dffCIO</a></p>
<p>— 即时中国大陆映像✊✊✊ (@o66071443) <a href="https://twitter.com/o66071443/status/887162853450436608">July 18, 2017</a></p></blockquote>
<blockquote class="translation"><p>Once installed with Jingwang, computers and mobiles in Xinjiang, would become electronic handcuffs.</p></blockquote>
<p>The Jingwang requirement may be a response to the city party committee’s instruction to crack down on the distribution of illegal content. All government branches and government and party affiliated institutions, including universities and research organizations, are obligated to follow instructions from the <a href="http://www.xjipc.cas.cn/xwzx/zhxw/201707/t20170705_4828450.html">Cybersecurity Office</a> on managing their computer and communication networks.</p>
<p>A recent report from Freedom House, a US-based human rights group, also <a href="https://freedomhouse.org/blog/china-s-newest-censorship-methods-display">touched on</a> the new surveillance practices:</p>
<blockquote><p>In Xinjiang, authorities in a district of the regional capital Urumqi issued a notice on June 27 instructing all residents and business owners to submit their &#8220;personal ID cards, cell phones, external drives, portable hard drives, notebook computers, and media storage cards&#8221; to the local police post for &#8220;registration and scanning&#8221; by August 1.</p>
<p>[&#8230;]</p>
<p>The goal is ostensibly to identify and purge any &#8220;terrorist videos,&#8221; but the action violates the privacy rights of Urumqi&#39;s three million residents and exposes them to punishment for a host of other possible offenses, including those related to peaceful religious or political expression.</p></blockquote>

	jQuery(document).ready(function($){

		 * Initialize attempt counter variable to

		var attemptCounter = 0;

		/**
		 * Add blind toggle to the trigger to slide the box out on click
		 */
		$(".email-share-form-trigger").click(function(){
			// Load form if it doesn't exist yet
			// Also handles toggling so that it slides AFTER content is loaded
			if ( 0 == $("#email-share-form").length ) {
				gvEsfAjaxLoadForm();
			// Otherwise just toggle the box (only used if form was already loaded)
			} else {
				$("#email-share-wrapper").toggle('blind');
			}
		 });

		/**
		 * Use AJAX to load the email form
		 *
		 * Depends on gv_esf_ajax_load_form() php function being hooked correctly
		 */
		function gvEsfAjaxLoadForm() {
			// Prepare ajax data with form fields to autofill
			var ajaxData = {
				action: 'gv_email_share_form_load',
				post_id: '24154',
				form_url: '/2017/07/19/chinas-xinjiang-residents-are-being-forced-to-install-surveillance-apps-on-mobile-phones/',
				from_name: '',
				from_email: ''
			}

			// Show the loading icon
			$('.esf-loading-icon').show();

			// Send ajax request
			$.ajax({
				type: 'POST',
				url: 'https://advox.globalvoices.org/wp-admin/admin-ajax.php',
				data: ajaxData,
				success: function(response) {
					// Insert response (presumed to be HTML of form) into wrapper
					$('#email-share-wrapper').append(response);

                    if (typeof grecaptcha != 'undefined') {
                        var holderId = grecaptcha.render('email-share-recaptcha-holder', {
                            'sitekey': '6LeusxoUAAAAACPdQ6XgmLNImULpa-gRylyMwpTZ',
                            'size': 'invisible',
                            'badge': 'inline',
                            'callback': function (recaptchaToken) {
                                gvEsfAjaxSend();
                                grecaptcha.reset(holderId);
                            },
                            'expired-callback': function () {
                                grecaptcha.reset(holderId);
                            }
                        });
                    }

					// Add click event for AJAX sending to submit button
					$('#gv_esf_ajaxsubmit').click(function(){
					    if (typeof grecaptcha != 'undefined') {
                            grecaptcha.execute(holderId);
                        } else {
                            gvEsfAjaxSend();
                        }
					});

					// Toggle slide the box
					$("#email-share-wrapper").toggle('blind');

					// Get the current scroll distance from the top of the window
					// var windowtop = $(window).scrollTop();

					// Fetch vertical position of container div
					// jquery ref http://api.jquery.com/offset/
					// var offsettop = $('#email-share-form-contents').offset().top;
					// Use container because it results in better position, can be changed to anything
					var offsettop = $('#email-share-wrapper').offset().top;
					console.log(offsettop);

			});

		/**
		 * ESF - Send AJAX request to send email from form
		 *
		 * Prepares and sends form data to WordPress AJAX where it will
		 * be handled and a reply with errors/sucess messages sent back.
		 */
		function gvEsfAjaxSend() {
			var ajaxData = {
				action: 'gv_email_share_form_send',
				gv_esf_post_id: $('#gv_esf_post_id').val(),
				gv_esf_to_3Rt8: $('#gv_esf_to_3Rt8').val(),
				gv_esf_from_R8h5i: $('#gv_esf_from_R8h5i').val(),

				gv_esf_from_0u5Rd: $('#gv_esf_from_0u5Rd').val(),
				email: $('#email').val(),
				gv_esf_message: $('#gv_esf_message').val(),
				gv_esf_counter: $('#gv_esf_counter').val()
			}

			// Show the loading icon
			$('#esf-sending-icon').show();

			$.ajax({
			  type: 'POST',
			  url: 'https://advox.globalvoices.org/wp-admin/admin-ajax.php',
			  data: ajaxData,
			  success: gvEsfAjaxSendResponseHandler
			});
		}

		/**
		 * Handle JSON response returned by Form AJAX Request.
		 *
		 * Should be called in gvEsfAjaxSend() as success of ajax request.
		 */
		function gvEsfAjaxSendResponseHandler(jsonResponse) {

			var output = '';

			// Make sure we recieved a response
			if (!jsonResponse) {
				output += '<p class="error">ERROR: NO AJAX RESPONSE</p>';
			}

			// Parse the response into an object
			// NOTE: new jQuery version means we don't need to parse JSON any more
			// var response = JSON.parse(jsonResponse);
			var response = jsonResponse;

			// Ghettoize error
			if (response.ghettoize) {
				gvAefGhettoize();
			}

			// If counter is too high then also ghettoize
			if (3 <= attemptCounter) {
				gvAefGhettoize();
			}
			attemptCounter++;

			// Email form error
			if (response.email_form_error) {
				output += '<p class="error">ERROR! Please fill out your email address, name and message, then resubmit</p>' ;
			}

			// Captcha Error
			if (response.captcha_error) {
				output += '<p class="error">ERROR! Incorrect response to test question. Please copy the letters and numbers exactly.</p>' ;
			}

			// Email Sent message
			if (response.email_sent) {

				// Clear any errors
				$('#email-share-form-messages').empty();
				// Clear the captcha field
				$('#email_address').val('');
				// Clear the recipient email field for next sending
				$('#gv_esf_to_3Rt8').val('');
				// Reset the counter in case they want to send another message for some reason.
				attemptCounter = 0;

				// Add success message to output
				output += '<p class="success">Email successfully sent</p>' ;
			}

			// Append any output to the messages div
			$('#email-share-form-messages').append(output);

			// Hide the loading icon
			$('#esf-sending-icon').hide();
		}
		/**
		 * Ghettoize this visitor (send them to a dead end error page)
		 *
		 *  Used to avoid bots retrying the form indefinitely.
		 *
		 * Depends on gv_email_ghetto() php function to serve ghetto
		 */
		function gvAefGhettoize() {
			var message = $('#gv_esf_message').val();
			var messageEscaped = escape(message);
			window.location = '?gv_email_ghetto=1\&email_text=' + messageEscaped;
		}
	});

		/**
		 * Fixed position sharing tools

		 * Makes sharing tools fixed position only when the viewport is mid-post
		 *

		 * When the window is resized, updated the fixed status
		 */
		$(window).resize(function () {			
			update_fixed_element(fixedParent, fixedElement, container, 100);
		});

		// performance-enhancing timer http://ejohn.org/blog/learning-from-twitter/#comment-391945
//		var fixedElementScrollTimer = 0;

		/**
		 * When window is scrolled update the fixed status
		 */
		$(window).scroll(function () {
		
			// Use a buffer so we don't call myCallback too often.
//			if (fixedElementScrollTimer) {
//				clearTimeout(fixedElementScrollTimer);
//			}
//			fixedElementScrollTimer = setTimeout(myCallback, 100);

			update_fixed_element(fixedParent, fixedElement, container, 100);
		});

		/**
		 * Update the position:fixed status of fixedElement inside fixedParent
		 * 
		 * Checks the position of the parent and element in relation to the viewport and
		 * makes fixedElement position:fixed; if it would otherwise have scrolled out
		 * of view.
		 * 
		 * Also disables the fixed position if the element is at the bottom of container, instead
		 * making fixedParent absolute positioned at the bottom of container. 
		 * 
		 * @param object fixedParent jQuery object with results of search for the container div for fixedElement (must be absolute positioned)
		 * @param object fixedElement jQuery object with results of search for the element that will sometimes be position:fixed
		 * @param integer topSpacing Vertical height in pixes of "padding" above fixed element when position:fixed;
		 */
		function update_fixed_element(fixedParent, fixedElement, container, topSpacing) {
			
			// get the distance the viewport has scrolled down the page 
			var scrollPosition = $(window).scrollTop();
	
			// get the distance from the top of screen to the top of the parent element
			fixedParentTop = fixedParent.offset().top;
			
			// Get the bottom position fixed element itself
			fixedElementBottom = fixedElement.offset().top + fixedElement.height();
			
			// Get the bottom position of the container
			containerBottom = container.offset().top + container.height();
						
			// Get the minimum scrollPosition at which we'll make fixedElement fixed
			// i.e. its natural position minus our padding/spacing so that we start early
			fixedScrollPosition = container.offset().top - topSpacing;
			
			// Get the minimum scrollPosition at which we'll stick fixedParent to the bottom of container
			// i.e. the point when fixedElement would be outside container
			bottomScrollPosition = containerBottom - fixedElement.height() - topSpacing;
			
//			console.log('fixedElementBottom:' + fixedElementBottom + ' containerBottom:' + containerBottom + ' fixedScrollPosition:' + fixedScrollPosition + ' scrollPosition:' + scrollPosition + '');
//			console.log('fixedElement.height:' + fixedElement.height());
//			console.log('fixedParent.height:' + fixedParent.height());
//			console.log('fixedParentTop:' + fixedParentTop);
//			console.log('fixedScrollPosition:' + fixedScrollPosition);
//			console.log('bottomScrollPosition:' + bottomScrollPosition);
//			console.log('fixedElementBottom:' + fixedElementBottom);
//			console.log('containerBottom:' + containerBottom);
			
//			fixedParent.css('background', 'purple');
//			fixedElement.css('background', 'pink');
//			container.css('background', 'orange');
						
			/**
			 * If the window scroll is less than the fixed starting position, set it to default positioning
			 */
			if (scrollPosition < fixedScrollPosition) {
				// Make it static (default)
				fixedElement.css('position', 'static');
				// Reset the parent's bottom in case it was set to zero
				fixedParent.css('bottom', 'auto');
				
				//console.log ('scrollPosition (' + scrollPosition + ') < fixedScrollPosition (' + fixedScrollPosition + ') -- make it top');
			
			/**
			 * If we have scrolled passed the bottom position (where fixedElement hits bottom of container) stick fixedParent to bottom
			 */
			}  else if (scrollPosition >= bottomScrollPosition) {
				// Set the element itself to static
				fixedElement.css('position', 'static');
				// Set the absolute-positioned parent to be at the bottom
				fixedParent.css('bottom', '0');

				//console.log ('fixedElementBottom (' + fixedElementBottom + ') >= containerBottom (' + containerBottom + ') -- make it bottom');
			
			/**
			 * Otherwise make fixedElement position:fixed so it can float up and down inside container
			 */
			} else {
				fixedElement.css('position', 'fixed');
				// Reset the absolute-positioned parent's bottom case it was set to 0
				fixedParent.css('bottom', 'auto');

				//console.log ('fixedElementBottom (' + fixedElementBottom + ') < containerBottom (' + containerBottom + ') -- make it fixed');
			};
		}

		 * @param string target_container jQuery selector for existing container that will hold new menu
		 * @param string target_css_class CSS class (no .) to add to new menu's NEW wrapper div
		 * @param string toggle_selector jQuery selector for the toggle button already inside menu content
		 * @param string menu_container_selector jQuery selector for new menu content container (below heading)
		 * @param string menu_content_selector jQuery selector for new menu content inside it's container
		 * @returns false
		 */
		function gv_clone_menu(source_container, target_container, target_css_class, toggle_selector, menu_container_selector, menu_content_selector) {
			source_container_html = $(source_container).html();
			
			if (source_container_html) {
				// Add the content to the new container wrapped in div.target_css_class
				$(target_container).append("<div class='" + target_css_class + " toggle-menu menu-closed'>" + source_container_html + "</div>");
				// Add .menu-toggle and an html arrow to the toggle
				$(target_container).find(toggle_selector).addClass('menu-toggle');
				$(target_container).find(toggle_selector).addClass('toggle-menu-heading');
				$(target_container).find(toggle_selector).append(" <span class='icon icon-caret-down'></span>");
				
				// Add .menu-content-container to the menu_container_selector
				$(menu_container_selector).addClass('toggle-menu-content-container');
				// Wrap the entire contents of the container in div.toggle-menu-content
				$(menu_container_selector).contents().wrapAll("<div class='toggle-menu-content'></div>");
			}
			return false;
		}

		/**
		 * Toggle the up/down font icon arrow inside parent_selector
		 * 
		 * @param string parent_selector jQuery selector for element that contains the span.icon 
		 */
		function gv_toggle_font_icon_arrow(parent_selector) {
			// Define the font icon classes to add to span
			icon_up_class = 'icon-caret-up';
			icon_down_class = 'icon-caret-down';
			
			// Get the .icon and make sure it exists
			arrow_icon = $(parent_selector).find('.icon');
			if (!arrow_icon.length) {
				return;
			}
				
			// Swap the icon classes
			if (arrow_icon.hasClass(icon_up_class)) {
				arrow_icon.removeClass(icon_up_class);
				arrow_icon.addClass(icon_down_class);
			} else if (arrow_icon.hasClass(icon_down_class)) {
				arrow_icon.removeClass(icon_down_class);
				arrow_icon.addClass(icon_up_class);
			}
		}
	/**
	 * Handles closing/opening/handlers for data-close elements.
	 */
	function gv_data_close() {
		var cookies = $.cookie();

		// Scan all data-close elements
		$('[data-close]').each(function(){
			var id = $(this).attr('data-close');

			// Hide the element if gv_hide_$id cookie is present
			if (typeof $.cookie('gv_hide_' + id) !== 'undefined') {
				$('#' + id).hide();
			// Explicitly show the element if the cookie doesn't exist
			} else if (typeof $.cookie('gv_hide_' + id) === 'undefined') {
				$('#' + id).show();
			}
		});

		// Add on click events to all data-close elements
		$('[data-close]').on('click', function() {
			var id = $(this).attr('data-close');

			$.cookie('gv_hide_' + id, true, { path: '/', expires: 30 });
			$('#' + id).hide();
		});
	}
	gv_data_close();
